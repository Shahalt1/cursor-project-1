Cursor Documentation

Welcome to the official documentation for Cursor IDE. This guide will help you get started with Cursor and make the most of its powerful features.

Installation

Cursor IDE is available for Windows, macOS, and Linux. Download the latest version from our website and follow the installation instructions for your operating system.

System Requirements

  • Windows 10 or later
  • macOS 10.15 or later
  • Ubuntu 18.04 or later
  • 4GB RAM minimum (8GB recommended)
  • 1GB free disk space

Quick Start

After installing Cursor, follow these steps to get started:

  1. Launch Cursor IDE from your applications menu
  2. Create a new project or open an existing one
  3. Start coding and experience the power of AI assistance
Note: Cursor automatically saves your work as you type, so you don't need to worry about losing your progress.

Keyboard Shortcuts

Here are some essential keyboard shortcuts to help you be more productive:

Ctrl+Space      AI code completion
Ctrl+P          Quick open file
Ctrl+Shift+P    Command palette
Ctrl+F          Find in file
Ctrl+Shift+F    Find in project
F5              Start debugging
F9              Toggle breakpoint
Ctrl+`          Toggle terminal

AI Code Completion

Cursor's AI-powered code completion is its most powerful feature. It understands the context of your code and offers intelligent suggestions as you type.

Using AI Completion

Simply start typing your code, and Cursor will suggest completions automatically. Press Tab to accept a suggestion, or continue typing to refine it.

You can also trigger AI completions manually by pressing Ctrl+Space at any point while coding.

Example

function calculateTotal(items) {
    // Cursor will suggest the following code:
    return items.reduce((total, item) => {
        return total + item.price * item.quantity;
    }, 0);
}

Real-time Collaboration

Cursor makes it easy to collaborate with your team in real-time. Multiple users can edit the same file simultaneously, with changes synced instantly.

Starting a Collaboration Session

To start collaborating:

  1. Click on the "Share" button in the top-right corner
  2. Copy the generated link and send it to your collaborators
  3. When they open the link, they'll join your session automatically

Debugging

Cursor includes a powerful debugger that helps you find and fix issues in your code.

Basic Debugging

To start debugging:

  1. Set breakpoints by clicking in the gutter next to line numbers
  2. Press F5 to start debugging
  3. Use the debug console to inspect variables and evaluate expressions
